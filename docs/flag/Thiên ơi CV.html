- tên module,
- mã thiết bị,
- nhà cung cấp,
- ngày nhận,

- hình ảnh trước lắp,
- phụ kiện đi kèm,
- vị trí lắp dự kiến.

Không được bóc niêm phong nếu chưa có ảnh intake.

### 5.2. Site Readiness

Kiểm tra hiện trường:

- mặt bằng khô ráo,
- nguồn điện ổn định,
- tiếp địa có đo kiểm,
- không đặt gần nguồn nhiệt bất thường,
- không đặt gần vùng nhiễu điện từ cao nếu chưa có che chắn,
- rack / enclosure cố định,
- đường dây không kéo căng,
- đầu nối không lỏng,
- không có dây tín hiệu đi chung bó với dây tải cao nếu chưa chống nhiễu.

Nếu không đạt:

`SITE_NOT_READY`

### 5.3. Physical Mounting

Yêu cầu:

- module cố định chắc,
- không ép cong board,
- không để dây đè lên khe tản nhiệt,
- đầu nối có strain relief,
- cổng service dễ tiếp cận,
- nhãn module đọc được sau khi lắp,
- vị trí module khớp sơ đồ field.

Nếu phát hiện sai vị trí:

`MOUNTING_REJECTED`

Trước khi cấp nguồn:

- đo continuity,
- đo ngắn mạch,
- đo phân cực,

- kiểm tra dây tín hiệu,
- kiểm tra dây nguồn,
- kiểm tra shielding,
- kiểm tra đầu cos / terminal,
- kiểm tra màu dây hoặc nhãn dây.

Không cấp nguồn khi chưa hoàn tất wiring verification.

### 5.5. First Power-On

Khi cấp nguồn lần đầu, bắt buộc ghi:

- thời điểm cấp nguồn,
- điện áp đầu vào,
- dòng khởi động,
- nhiệt độ môi trường,
- âm thanh bất thường nếu có,
- mùi khét nếu có,
- đèn báo,
- log thiết bị nếu có.

Nếu phát sinh bất thường:

`POWER_ON_ABORTED`

### 5.6. First-Light / First-Signal

Đối với module quang / màu / cảm biến:

- phát tín hiệu thử,
- nhận tín hiệu thử,
- đo mức nhiễu nền,
- ghi nhận độ trễ,
- ghi nhận khoảng cách,
- ghi nhận góc lệch,
- ghi nhận độ suy hao,

Nếu không nhận được tín hiệu:

`FIRST_SIGNAL_FAILED`

## 7. Red Zone Abort

Bắt buộc dừng lắp đặt nếu có một trong các lỗi:

1. Nguồn vượt ngưỡng cho phép.
2. Dòng khởi động bất thường.
3. Mùi khét.
4. Nhiệt tăng nhanh.
5. Vỏ thiết bị rò điện.
6. Đầu nối nóng bất thường.
7. Tín hiệu laser / IR không kiểm soát hướng phát.
8. Cảm biến trả tín hiệu nhiễu vượt baseline.
9. Sai model so với Định Mức Quy Chuẩn.
10. Serial không khớp hồ sơ.
11. Nhà cung cấp tráo module.
12. Không có người chịu trách nhiệm nghiệm thu.

Trạng thái:

`RED_ZONE_ABORT`

import numpy as np
import matplotlib.pyplot as plt
from pathlib import Path

# Render-language probe for the current structural proposal.
# This is a numerical/visual probe, not a final physics proof.

n = 4
lambda0 = 387.0
gamma0 = np.log(2) / 24.0
T = 24.0
gamma_n = gamma0 * (n + 1)

# Grid in a normalized surface plane. Units are symbolic, not physical meters.
N = 700
extent = 10.0
x = np.linspace(-extent, extent, N)
y = np.linspace(-extent, extent, N)
X, Y = np.meshgrid(x, y)
R = np.sqrt(X**2 + Y**2) + 1e-9
Theta = np.arctan2(Y, X)

# KEY / raw scale. Kept separate from normalized state.
key_raw = lambda0 * n**2

# Orbital-like shape:
# n=4 corresponds to an f-like shell in the current metaphor, so use l=3.
# This is a real-valued orbital analog, not a hydrogenic claim.
l = 3
radial = (R / (n + 1))**l * np.exp(-R / (n + 1))
angular = np.cos(l * Theta)
phi = radial * angular

# Bond product: three multiplicative "bond satellites".
# beta > 1 near bond centers; ~1 elsewhere.
bond_centers = [(-4.0, 0.0), (3.0, 3.5), (3.0, -3.5)]
bond = np.ones_like(R)
for cx, cy in bond_centers:
    d2 = (X - cx)**2 + (Y - cy)**2
    beta = 1.0 + 0.32 * np.exp(-d2 / 3.0)
    bond *= beta

# Stress attenuation: directional stress from +x side and upper-right quadrant.
stress_1 = 0.38 * np.exp(-((X - 6.0)**2 + Y**2) / 18.0)
stress_2 = 0.25 * np.exp(-((X - 3.0)**2 + (Y - 5.0)**2) / 10.0)
attenuation = np.exp(-(stress_1 + stress_2))

# Decay at T = 24h.
decay = np.exp(-gamma_n * T)

# Raw render field and normalized intensity.
F_raw = key_raw * phi * decay * bond * attenuation
intensity = np.abs(F_raw)**2
intensity_norm = intensity / (intensity.max() + 1e-12)

# Numerical diagnostics.
mass = np.trapz(np.trapz(intensity_norm, x), y)
peak_idx = np.unravel_index(np.argmax(intensity_norm), intensity_norm.shape)
peak_xy = (float(X[peak_idx]), float(Y[peak_idx]))
center_value = float(intensity_norm[N//2, N//2])
decay_pct = float(decay * 100.0)

# Save image.
out = Path("/mnt/data/nauion_render_probe_n4_t24.png")
plt.figure(figsize=(8, 8))
plt.imshow(intensity_norm, origin="lower", extent=[-extent, extent, -extent, extent])
plt.title("NauionOS Render Probe | n=4, T=24h | normalized |R|²")
plt.xlabel("x / normalized surface coordinate")
plt.ylabel("y / normalized surface coordinate")
plt.colorbar(label="normalized intensity")
plt.tight_layout()
plt.savefig(out, dpi=180)
plt.close()

print("Probe complete")
print(f"n = {n}")
print(f"lambda0*n^2 = {key_raw:.2f} nm raw scale")
print(f"gamma(n) = {gamma_n:.6f} per hour")
