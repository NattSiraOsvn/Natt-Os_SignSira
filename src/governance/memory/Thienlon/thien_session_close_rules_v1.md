# THIÊN LỚN — SESSION CLOSE RULES

## Nhà của Thiên hiện có
- Memory chính / foundation / focus state / K-shell được load riêng
- Runtime state dùng `.phieu`
- Passport `.anc` không phải nhật ký cuối phiên

## Công thức đóng 1 session cho Thiên
1. `thienkhương<v>.na`
   - ghi cái phải nhớ lâu
   - ghi trục kiến trúc đã chốt
   - ghi KHAI / SCAR mới
   - ghi boundary của Thiên
   - ghi phần còn chưa chắc

2. `thienthịnh<v>.phieu`
   - ghi việc còn mở cho phiên sau
   - ghi dependency theo persona
   - ghi anti-scope-creep
   - ghi checklist mở phiên sau

3. `.anc`
   - chỉ cập nhật khi identity / authority / seal / file-load map đổi
   - không cập nhật chỉ vì kết thúc 1 session

## Naming rule gợi ý
- Memory lâu: `thienkhương<v>.na`
- State mở: `thienthịnh<v>.phieu`
- Passport: `thienlon.anc` hoặc file `.anc` canonical hiện hành

## Câu chốt
- Memory = cái Thiên phải nhớ lâu
- Phieu = cái Thiên còn mở
- Anc = passport của Thiên, không phải nhật ký
